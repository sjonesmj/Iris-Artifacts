# Iris Artifacts

A mobile-friendly GitHub Pages site for personal planning artifacts: gardening, lists, budgeting, and tracking.

## Structure

```text
index.html                 # Home page
pages/garden/index.html    # Gardening hub
pages/lists/index.html     # General lists hub
pages/budget/index.html    # Budget hub
pages/tracker/index.html   # Tracking hub
assets/css/styles.css      # Shared responsive styles
assets/js/app.js           # Mobile menu and tiny helpers
```

## GitHub Pages

This repository is configured to publish from the `main` branch root (`/`).
